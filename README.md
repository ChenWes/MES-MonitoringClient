# MES Monitoring Client
系统分为两块（UI界面及后台数据上传服务）


## 使用到的技术点
* C#
* Timer（定时器）
* Thread（线程）
* ServialPort（串口编程）
* MongoDB.Driver（Mongodb C#客户端）
* RabbitMQ.Client（RabbitMQ C# 客户端）
